# projeto-android-atv-fisica

## Projeto de Disciplina 

Criação de uma rede onde usuários compartilham dados de atividades físicas utilizando sensores de smartphones.

## Tecnologias:
- Android API 31
- Sensores
  - Acelerômetro
  - Frequência Cardíaca
  - Barômetro
  - GPS
- Firebase Authentication
  - Google
  - Email Provider
- Cloud Firestore
- MPAndroidChart
